# 🎮 Simon Says Game

A fun and interactive memory game built using **HTML**, **CSS**, and **JavaScript**.  
Test your memory skills by repeating the sequence of colors in the correct order — as the level increases, so does the challenge!

## 🚀 Demo

You can play the game locally by opening `index.html` in your browser.

## 🧩 How to Play

1. Press **Start** (or any key) to begin the game.  
2. The game will flash a color — **remember it!**  
3. Click or press the color in the **same order** as shown.  
4. Each round adds a new color to the sequence.  
5. If you click the **wrong color**, the game ends.  
6. Try to get the **highest score** possible!

## 🕹️ Features

- Colorful, animated buttons  
- Increasing difficulty each round  
- Sound effects for feedback (optional)  
- Game over and restart functionality  
- Clean and responsive design

## 🧠 Technologies Used

- **HTML5** – structure  
- **CSS3** – styling and animations  
- **JavaScript (ES6)** – game logic and interactivity
